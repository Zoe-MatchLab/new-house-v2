## 1. 背景与目标

### 1.1 模块定位

消息通知是**跨模块基础设施**，为其他模块提供统一的通知触达能力。

### 1.2 需求背景

**为什么要做消息通知？**

- 各模块需要通知相关人员
- 需要统一的通知渠道（小程序+微信）
- 需要区分通知优先级

### 1.3 目标

| 目标   | 说明              |
| ---- | --------------- |
| 统一通知 | 各模块的通知统一通过本模块发送 |
| 多渠道  | 小程序内部 + 微信同步    |
| 优先级  | 支持普通/重要/紧急三种优先级 |

***

## 2. 核心功能

### 2.1 三大核心功能

| # | 核心功能     | 说明           |
| - | -------- | ------------ |
| 1 | **通知发送** | 各模块触发通知，统一发送 |
| 2 | **通知接收** | 小程序/微信接收通知   |
| 3 | **通知管理** | 查看通知、设置免打扰   |

***

## 3. 通知类型定义

### 3.1 通知类型汇总

| 通知类型         | 来源模块 | 接收人      | 触发时机       | 优先级 |
| ------------ | ---- | -------- | ---------- | --- |
| **任务催办（轻度）** | 任务管理 | 经纪人      | 任务结束前24h   | 普通  |
| **任务催办（中度）** | 任务管理 | 经纪人      | 任务结束前12h   | 重要  |
| **任务催办（重度）** | 任务管理 | 经纪人      | 任务结束前2h    | 紧急  |
| **手动催办**     | 任务管理 | 经纪人      | 区域经理手动点击催办 | 紧急  |
| **任务超时提醒**   | 任务管理 | 经纪人      | 任务到期时间到    | 紧急  |
| **任务完成通知**   | 任务管理 | 区域经理     | 任务完成       | 普通  |
| **内容生成完成**   | 路书任务 | 经纪人      | AI内容生成完成   | 普通  |
| **新盘上线通知**   | 新盘管理 | 区域经理     | 楼盘上线       | 重要  |
| **新盘下线通知**   | 新盘管理 | 区域经理+经纪人 | 楼盘下线       | 紧急  |
| **录音分析完成**   | 录音分析 | 经纪人      | AI分析完成     | 普通  |

### 3.2 优先级定义

| 优先级 | 说明        | 展示效果   |
| --- | --------- | ------ |
| 普通  | 一般性通知     | 静默推送   |
| 重要  | 需要关注的通知   | 强提醒    |
| 紧急  | 需要立即处理的通知 | 强提醒+震动 |

***

## 4. 通知渠道

### 4.1 渠道定义

| 渠道    | 说明             |
| ----- | -------------- |
| 小程序内部 | 主要通知渠道，在小程序内展示 |
| 微信    | 同步推送至微信        |

### 4.2 推送策略

| 优先级 | 小程序 | 微信     |
| --- | --- | ------ |
| 普通  | ✅   | ❌      |
| 重要  | ✅   | ✅      |
| 紧急  | ✅   | ✅（强提醒） |

***

## 5. 通知内容结构

### 5.1 通知字段

| 字段   | 说明                              |
| ---- | ------------------------------- |
| 通知ID | 唯一标识                            |
| 标题   | 通知简述                            |
| 内容   | 详细说明                            |
| 优先级  | 普通/重要/紧急                        |
| 来源模块 | 触发通知的模块                         |
| 关联ID | 关联数据的ID（如task\_id、building\_id） |
| 跳转链接 | 点击后跳转的页面                        |
| 创建时间 | 通知创建时间                          |
| 已读状态 | 已读/未读                           |

### 5.2 通知内容示例

```
【任务催办-重度】
任务「xxx楼盘探盘」即将到期（2小时后），请尽快完成！
[查看任务]
```
