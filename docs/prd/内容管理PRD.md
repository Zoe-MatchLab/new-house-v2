## 1. 背景与目标

### 1.1 模块定位

内容管理是**内容资产管理模块**，用于统一管理任务模块生成的各类营销内容（小红书文案、封面图、短视频脚本、视频等）。

> 原"物料管理"模块升级为"内容管理"，更强调内容的生成与管理属性。

### 1.2 需求背景

**为什么要做内容管理？**
- 内容生成分散在各任务中，无统一管理入口
- 无法跨任务复用已生成的内容
- 难以追踪内容使用情况

**当前问题**
- 内容只在单个任务内可见
- 无法批量管理/导出内容
- 内容与原始素材混在一起，难以区分

### 1.3 目标

| 目标 | 说明 |
|-----|------|
| 统一管理 | PC端查看/导出所有生成内容 |
| 渠道分类 | 按渠道（小红书/公众号/短视频）展示 |
| 素材关联 | 内容关联原始素材，便于追溯 |
| 复用能力 | 支持内容复制到其他任务 |

## 2. 核心功能

### 2.1 三大核心功能

| # | 核心功能 | 说明 |

## 5. 小程序端功能

### 5.1 入口

- **我的 → 工具 → 内容管理**
- **任务完成 → 查看内容**（跳转到内容详情）

### 5.2 页面清单

| 页面 | 功能 |
|-----|------|
| 内容列表 | 查看已生成内容 |
| 内容详情 | 查看各渠道内容 |
| 内容生成 | 复用任务模块的生成流程 |

### 5.3 内容生成流程（复用任务模块）

> 内容生成能力由任务模块提供，内容管理模块负责查看和管理。

```
任务模块 → 内容生成流程
    ├── 阶段1：定位校验（楼盘信息确认）
    ├── 阶段2：素材上传（上传图片/视频素材）
    ├── 阶段3：内容生成（AI生成各渠道内容）
    │       ├── 小红书文案
    │       ├── 小红书封面图
    │       ├── 公众号文案
    │       └── 短视频脚本
    └── 阶段4：确认提交
            ├── 文案编辑/重新生成
            ├── 封面图选择
            ├── 视频生成
            └── 提交到内容管理
```

### 5.4 内容查看页面

| 功能 | 说明 |
|-----|------|
| 楼盘信息 | 显示关联的楼盘 |
| 渠道Tab | 切换查看不同渠道内容 |
| 内容预览 | 查看文案/图片/视频 |
| 操作 | 复制/下载/分享 |

## 7. 权限模型

### 7.1 数据可见性

| 用户类型 | PC端 | 小程序端 |
|---------|-----|---------|
| 超级管理员 | 全部内容 | - |

| 角色 | 查看 | 新增 | 编辑 | 删除 | 导出 |

| 区域经理 | 团队内容 | - | - | - | ✓ |
