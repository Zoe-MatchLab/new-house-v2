## 一、业务概述

### 1.1 模块定位

任务中心是房产AI作业平台的**统一任务入口**，面向所有用户（经纪人、区域经理、管理员）提供任务查看、执行和管理的能力。

### 1.2 用户角色

| 角色 | 说明 | 权限范围 |
|-----|------|---------|
| 经纪人 | 执行被分配的任务 | 查看自己被分配的任务、执行打卡、提交审核 |
| 区域经理 | 管辖团队任务 | 查看团队所有任务、催办、统计 |
| 管理员/超管 | 全局管理 | 查看所有任务、配置、统计 |

### 1.3 核心价值

| 价值点 | 说明 |
|-------|------|
| 统一入口 | 所有任务类型（探盘、话术、录音、文案）统一管理 |
| 移动优先 | 经纪人可随时随地执行任务 |
| 实时同步 | 任务状态实时更新，管理者随时掌握进度 |
| 智能提醒 | 多级提醒机制，确保任务及时完成 |

## 三、核心对象设计

### 3.1 任务（Task）

```typescript
interface Task {
  // 基本信息
  id: string;                    // 任务ID，格式：TASK_xxx
  name: string;                 // 任务名称
  type: TaskType;                // 任务类型
  status: TaskStatus;            // 任务状态
  priority: Priority;             // 优先级：P0/P1/P2
  
  // 关联信息
  buildingId: string;           // 关联楼盘ID
  buildingName: string;          // 关联楼盘名称
  creatorId: string;              // 创建人ID
  creatorName: string;           // 创建人姓名
  
  // 时间信息
  createdAt: string;             // 创建时间
 下发At: string;                 // 下发时间
  startAt: string;               // 开始时间
  deadlineAt: string;             // 截止时间
  
  // 执行信息
  executors: Executor[];           // 执行人列表
  notifiers: Notifier[];          // 通知人列表
  
  // 配置信息
  config: TaskConfig;              // 任务配置（点位、提醒等）
  
  // 产出信息
  deliverables: Deliverable[];     // 产出物（文案、封面等）
}

enum TaskType {
  探盘任务 = 'patrol',           // 探盘任务
  话术训练 = 'script',           // 话术训练
  录音分析 = 'recording',        // 录音分析
  文案生成 = 'copywriting',      // 文案生成
  自定义 = 'custom'              // 自定义任务
}

enum TaskStatus {
  草稿 = 'draft',               // 草稿
  已下发 = 'distributed',        // 已下发
  进行中 = 'in_progress',        // 进行中
  已完成 = 'completed',           // 已完成
  已超时 = 'overdue',            // 已超时
  已停止 = 'stopped'             // 已停止
}
```

### 3.2 执行人（Executor）

```typescript
interface Executor {
  userId: string;                // 用户ID
  userName: string;               // 用户姓名
  avatar: string;                  // 头像URL
  status: ExecutorStatus;          // 执行状态
  progress: number;               // 进度 0-100
  checkpoints: Checkpoint[];        // 打卡点完成情况
  startedAt: string;               // 开始执行时间
  completedAt: string;            // 完成时间
}

enum ExecutorStatus {
  待接收 = 'pending',            // 等待接受
  已接收 = 'received',            // 已接受
  进行中 = 'in_progress',        // 执行中
  已完成 = 'completed',           // 已完成
  已拒绝 = 'rejected'             // 已拒绝
}
```

### 3.3 打卡点（Checkpoint）

```typescript
interface Checkpoint {
  id: string;                    // 打卡点ID
  name: string;                  // 打卡点名称
  location: Location;             // 位置信息
  type: CheckpointType;            // 必打卡/选打卡
  uploadTypes: UploadType[];       // 上传类型：拍照/视频/文字
  status: CheckpointStatus;       // 打卡状态
  proof: Proof;                   // 打卡凭证
  checkedAt: string;              // 打卡时间
}

enum CheckpointType {
  必打卡 = 'required',           // 必须完成
  选打卡 = 'optional'            // 可选完成
}

enum CheckpointStatus {
  待打卡 = 'pending',            // 待打卡
  已打卡 = 'checked',            // 已打卡
  已超时 = 'overdue'             // 已超时
}

interface Proof {
  photos?: string[];              // 照片凭证
  video?: string;                 // 视频凭证
  text?: string;                  // 文字说明
  location?: Location;            // 打卡位置
}
```

interface User {

## 四、能力模块

### 4.1 任务创建

| 功能 | 说明 | 入口 |
|-----|------|-----|
| PC端创建 | 管理员在PC后台创建任务 | PC管理后台 → 任务管理 → 创建任务 |
| 任务类型选择 | 探盘/话术/录音/文案/自定义 | 创建任务表单 |
| 楼盘关联 | 选择任务关联的楼盘 | 创建任务表单 |
| 执行人指定 | 选择执行人或团队 | 创建任务表单 |
| 提醒配置 | 设置轻度/中度/重度提醒 | 创建任务表单 |
| 点位配置 | 设置打卡点位（探盘任务） | 创建任务表单 |

### 4.2 任务分发

| 功能 | 说明 | 触发条件 |
|-----|------|---------|
| 自动下发 | 创建后自动下发至执行人 | 管理员点击"下发" |
| 通知执行人 | 小程序推送通知 | 任务下发时 |
| 任务提醒 | 多级提醒（轻度/中度/重度） | 到达提醒时间 |

### 4.3 任务执行

| 功能 | 说明 | 入口 |
|-----|------|-----|
| 任务查看 | 执行人查看被分配的任务 | 小程序首页 |
| 打卡操作 | 执行人到现场拍照/视频/文字打卡 | 任务详情页 |
| 进度查看 | 实时查看任务完成进度 | 任务详情页 |
| 审核提交 | 完成后提交AI审核 | 任务详情页 |

### 4.4 数据回收

| 数据类型 | 说明 | 产出 |
|---------|-----|------|
| 打卡数据 | 执行人打卡记录、凭证 | 照片、视频、位置 |
| 内容数据 | AI生成的文案、封面 | 多渠道适配内容 |
| 审核数据 | AI审核结果 | 合规判定、问题标注 |
| 统计报表 | 任务完成率、超时率等 | 数据看板 |

### 7.2 权限说明

| 角色 | 数据范围 | 可执行操作 |
|-----|---------|-----------|
| **超级管理员** | 全局所有数据 | 创建/编辑/删除/下发/配置/统计 |
| **管理员** | 全局所有数据 | 创建/编辑/删除/下发/统计 |

### 7.3 权限控制点

| 控制点 | 实现方式 |
|-------|---------|
| 任务列表可见性 | 按角色过滤：经纪人只看自己被分配的任务 |
| 任务详情可见性 | 按参与关系：创建人、执行人、通知人可看 |
| 操作按钮可见性 | 按角色显示：只有管理员可见"编辑/删除" |
| 数据导出权限 | 仅管理员和区域经理可导出 |
