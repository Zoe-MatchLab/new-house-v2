---

## 1. 概述

### 1.1 设计目标

为任务管理模块提供统一的PC端管理界面，支持超管和区域经理查看、追踪、管理所有任务。

### 1.2 设计原则

| 原则 | 说明 |
|-----|------|
| 效率优先 | 批量操作、快速筛选、一目了然 |
| 清晰分层 | 按状态/类型/优先级分层展示 |
| 操作明确 | 每个状态对应明确可执行操作 |
| 只读优先 | PC端只提供查看功能 |

### 1.3 页面清单

| 页面名称 | 页面ID | 类型 | 说明 |
|---------|-------|-----|-----|
| 任务列表 | task-list | 列表页 | 主数据展示页 |
| 任务详情 | task-detail | 详情页 | 任务完整信息 |
| 任务创建 | task-create | 创建页 | 创建任务（后续版本） |
| 任务追踪 | task-track | 统计页 | 团队任务完成情况（后续版本） |

#### 筛选器定义

| 筛选项 | 类型 | 选项/说明 |
|-------|------|----------|
| 关键词搜索 | 文本输入 | 搜索：任务名称、楼盘名称 |
| 任务类型 | 下拉 | 全部 / 探盘任务 / 话术训练任务 / 录音分析任务 / 文案生成任务 / 自定义任务 |
| 任务状态 | 下拉 | 全部 / 草稿 / 进行中 / 已完成 / 已超时 / 已停止 |
| 创建人 | 下拉 | 全部 / 我的 / 指定人员 |
| 执行人 | 搜索 | 输入人员姓名联想搜索 |
| 时间范围 | 日期选择 | 创建时间 / 结束时间（两个日期选择器） |

#### 表格列定义

| 列名 | 列宽 | 说明 |
|-----|-----|------|
| 勾选 | 40px | 批量选择 |
| 展开 | 40px | 子任务展开按钮 |
| 任务名称 | 200px | 任务标题 |
| 任务类型 | 100px | 探盘任务/话术训练/录音分析/文案生成/自定义任务 |
| 任务状态 | 80px | 主任务状态 |
| 完成进度 | 70px | X/Y（已完成执行人数/总执行人数） |
| 创建人 | 80px | 任务创建人姓名 |
| 执行人 | 100px | 多个用气泡窗展示（>3个） |
| 创建时间 | 80px | 创建时间 |
| 结束时间 | 80px | 截止时间 |
| 提醒时间 | 80px | 催办提醒时间 |
| 操作 | 60px | 查看按钮 |

#### 子任务展示

| 展示方式 | 说明 |
|---------|------|
| 折叠态 | 显示"共X个子任务" |
| 展开态 | 显示子表格 |

**子表格字段**：

| # | 字段 | 说明 |
|---|-----|------|
| 1 | 任务名称 | 子任务名称 |
| 2 | 任务类型 | 子任务类型 |
| 3 | 任务状态 | 状态 |
| 4 | 操作 | 查看（仅查看） |

#### 执行人展示

| 数量 | 展示方式 |
|-----|---------|
| ≤3个 | 直接展示姓名 |
| >3个 | 鼠标悬停气泡窗展示 |

#### 状态标签样式

| 状态 | 标签样式 |
|-----|---------|
| 草稿 | 灰色标签 |
| 进行中 | 橙色标签 |
| 已完成 | 绿色标签 |
| 已超时 | 红色标签 |
| 已停止 | 灰色标签 |

#### 操作按钮

| 页面状态 | 操作按钮 |
|---------|---------|
| 任意状态 | 查看 |

## 4. 组件库

### 4.1 通用组件

| 组件 | 状态 |
|-----|-----|
| Button | 默认/悬停/点击/禁用/加载中 |
| Input | 默认/聚焦/错误/禁用 |
| Select | 默认/展开/选中/禁用 |
