## 1. 概述

### 1.1 设计目标

为任务管理模块提供统一的PC端管理界面，支持超管和区域经理查看、追踪、管理所有任务。

### 1.2 设计原则

| 原则 | 说明 |
|-----|------|
| 效率优先 | 批量操作、快速筛选、一目了然 |
| 清晰分层 | 按状态/类型/优先级分层展示 |
| 操作明确 | 每个状态对应明确可执行操作 |

### 1.3 页面清单

| 页面名称 | 页面ID | 类型 | 说明 |
|---------|-------|-----|-----|
| 任务列表 | task-list | 列表页 | 主数据展示页 |
| 任务详情 | task-detail | 详情页 | 任务完整信息 |
| 任务创建 | task-create | 创建页 | 创建探盘任务（后续版本） |
| 任务追踪 | task-track | 统计页 | 团队任务完成情况 |

│  │ └──────────┘ └──────────┘ └──────────┘ └──────┘ │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ 数据表格                                              │    │
│  │ ┌───┬─────────┬──────┬──────┬─────┬────────┬────┐ │    │
│  │ │勾选│ 楼盘名称 │ 执行人│状态  │优先级│截止时间│操作│ │    │
│  │ ├───┼─────────┼──────┼──────┼─────┼────────┼────┤ │    │

#### 组件清单

| 组件名 | 组件ID | 用途 | 状态说明 |
|-------|-------|-----|---------|
| 筛选器 | filter | 任务类型/状态/创建人筛选 | 默认/选中 |
| 搜索框 | search-input | 楼盘名称/执行人搜索 | 默认/聚焦/有内容 |
| 数据表格 | data-table | 任务列表展示 | 默认/加载中/空数据 |
| 状态标签 | status-tag | 任务状态展示 | 待接受/进行中/已完成/已超时 |
| 优先级标签 | priority-tag | 优先级展示 | 紧急/高/中/低 |
| 操作按钮 | action-btn | 操作入口 | 查看/编辑/删除/催办 |
| 分页组件 | pagination | 分页导航 | 默认/禁用/加载中 |
| 批量操作栏 | batch-bar | 批量操作入口 | 隐藏/显示 |

#### 状态说明

| 页面状态 | 说明 |
|---------|-----|
| 默认状态 | 正常展示筛选器 + 表格数据 |
| 加载中 | 表格显示loading骨架 |
| 空数据 | 表格显示空状态插图+文案 |
| 搜索无结果 | 显示"未找到匹配的任务" |

#### 操作按钮与状态对应

| 任务状态 | 可显示按钮 | 权限说明 |
|---------|-----------|---------|
| 待接受 | 查看、编辑、删除 | 创建者可操作 |
| 进行中 | 查看、暂停、催办 | 创建者可操作 |
| 已超时 | 查看、重新分配、延期 | 创建者可操作 |
| 已完成 | 仅查看 | 只读 |

#### 交互说明

| 交互 | 行为 |
|-----|------|
| 点击筛选器 | 下拉展示选项，选择后自动筛选 |
| 点击搜索 | 实时搜索，回车确认 |
| 点击表格行 | 打开任务详情 |
| 点击操作按钮 | 执行对应操作（编辑/删除/催办） |
| 点击批量勾选 | 底部批量操作栏动画弹出 |
| 点击批量操作 | 执行批量操作 |

## 6. 原型说明

### 6.1 原型入口

- PC任务列表：`../prototypes/pc/task/list.html`
- PC任务详情：`../prototypes/pc/task/detail.html`
- PC任务创建：`../prototypes/pc/task/create.html`
- PC任务追踪：`../prototypes/pc/task/track.html`

### 6.2 交互说明

| 页面 | 交互说明 |
|-----|---------|
| list.html | 状态筛选、搜索、批量操作、催办 |
| detail.html | 返回、查看关联数据、催办操作 |
| create.html | 楼盘选择、执行人选择、时间设置 |
| track.html | 统计数据、经纪人完成率 |

### 6.3 原型状态覆盖

| 页面 | 覆盖状态 |
|-----|---------|
| list.html | 待接受/进行中/已完成/已超时、催办轻度/中度/重度 |
| detail.html | 待接受/进行中/已完成/已超时 |
| create.html | 新建/编辑 |
| track.html | 正常数据/空数据 |
