# 房产AI作业平台 - 领导汇报材料

> **在线预览**
> - 小程序端：https://wxapp-seven.vercel.app
> - PC端：https://pc-five-sepia.vercel.app

---

## 一、项目概述

**房产AI作业平台**是一套帮助房产经纪人提升工作效率的智能工具系统，包含：

| 端 | 设备 | 页面数 | 链接 |
|---|-----|-------|------|
| 小程序端 | 手机/微信 | 24页 | https://wxapp-seven.vercel.app |
| PC端 | 电脑浏览器 | 17页 | https://pc-five-sepia.vercel.app |

---

## 四、核心功能流程

### 4.1 探盘任务完整流程

```
创建任务 → 任务下发 → 执行打卡 → AI审核 → 文案生成 → 封面生成 → 渠道发布
   │          │          │          │          │          │          │
   ▼          ▼          ▼          ▼          ▼          ▼          ▼
  管理员    执行人    执行人    系统AI    系统AI    系统AI    多渠道
  创建      接收      完成打卡  自动审核  生成文案  生成封面  一键发布
```

### 4.2 关键交互

| 功能 | 描述 |
|-----|------|
| 任务下发 | 管理员创建任务后自动/手动下发至执行人 |
| 打卡执行 | 执行人到现场拍照/视频/文字打卡 |
| AI审核 | 系统自动审核打卡内容合规性 |
| 文案生成 | 基于楼盘信息生成多渠道适配文案 |
| 封面生成 | AI生成小红书/朋友圈等封面图 |
| 一键发布 | 同时发布至多个社交平台 |
