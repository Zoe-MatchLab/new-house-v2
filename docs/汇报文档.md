# 新房AI作业平台V2 评审汇报

> 日期：2026-04-15
> 状态：📝待评审

## 一、项目概述

### 1.1 产品定位

新房AI作业平台V2是一款面向房地产经纪人的智能作业工具，通过AI能力提升经纪人专业能力和作业效率，同时为管理层提供数据支撑。

### 1.2 核心价值

| 价值点 | 说明 |
|-------|------|
| 快速了解楼盘 | 81字段结构化，图文并茂 |
| AI话术训练 | 智能对话练习，实时反馈 |
| 自动生成内容 | AI生成文案配图，降低成本 |
| 数据驱动管理 | 实时看板，掌握团队状态 |

### 1.3 目标用户

| 用户 | 使用端 | 核心诉求 |
|-----|-------|---------|
| 经纪人 | 小程序 | 执行任务、提升能力、管理客户 |
| 区域经理 | PC+小程序 | 任务下发、数据查看、团队管理 |
| 超级管理员 | PC | 系统配置、跨区管理 |

## 三、核心流程

### 3.1 探盘任务流程

```
PC端                    小程序端
   │                        │
   ▼                        │
创建任务 ──────────────────►│
   │                        │
   ▼                        ▼
规则配置 ──────────────────►│
   │                        │
   ▼                        ▼
选定人员 ──────────────────►│ 收到通知
   │                        │
   ▼                        ▼
确认下发                 执行任务
                          │
                          ▼
                    定位打卡 → 点位采集 → AI生成 → 确认发布
```

### 3.2 录音分析流程

```
选择楼盘 → 选择客户 → 上传录音 → AI分析 → 查看结果 → 改进提升
```

### 3.3 话术训练流程

```
选择场景 → AI客户模拟 → 对话练习 → 实时反馈 → 训练报告
```

## 四、AI能力

### 4.1 AI能力矩阵

| 能力 | 应用场景 | 技术规格 |
|-----|---------|---------|
| ASR语音转文字 | 录音分析 | 支持30分钟内音频，精度≥95% |

| AI对话练习 | 话术训练 | 流式输出，5种客户画像 |

|-----|------|------|
| 产品熟悉度 | 30% | 楼盘信息准确率、卖点覆盖度 |
| 需求挖掘 | 25% | 追问轮次、需求确认覆盖率 |
| 应变能力 | 25% | 异议处理覆盖率、响应延迟 |
| 成交意识 | 20% | 逼定时机、邀约技巧 |
